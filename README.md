## Creating a new Project

Make sure you have npm and node and create-react-app tool

```
create-react-app project-name
```

OR

```
git clone https://github.com/nonuabi/react-slack-clone
```

## Starting a project

```
npm start
```

## Project Structure

```
SRC
|───components
|   └───App.js
|   └───index.js
|   └───SignIn.js

```
